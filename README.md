## Como compilar e executar

### Requisitos
- Um compilador C (ex.: **GCC** ou **Clang**)

### Compilando um exercício (exemplo com GCC)

Dentro da pasta do projeto:

### Linux:
```bash
gcc Lista-01/ex01.c -o ex01
./ex01
```
### Windows:
```bash
gcc .\Lista-01\ex01.c -o ex01.exe
.\ex01.exe
```
